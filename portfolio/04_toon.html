Project Toon Shading

3D Graphics

Project Overview

The D04ToonShading project was an assignment for the CS250 Computer Graphics class at Digipen, building a toon shading framework that augmented custom_range_toon.frag, uniform_range_toon.frag, and gradient_texture_toon.frag with adaptive anti-aliasing and linear fog effects.

Tasks Completed

  • Implemented create_trefoil in Mesh.cpp.
  • toon.vert: Computes model→view→projection transformation, passes view space normals via a 3x3 normal matrix, and passes view position for fog effects.
  • Implemented three files: 01_custom_range_toon.frag, 02_uniform_range_toon.frag, 03_gradient_texture_toon.frag.
  • Anti-aliasing: Adaptively smoothed diffuse bands and specular edges using fwidth() and smoothstep().
  • Linear Fog: Uniformly applied view-space depth blending to all three Toon modes to achieve atmospheric depth cuing.

Reflection

The biggest issue I encountered while working on the project was that anti-aliasing was not applied properly. This was fixed by adding conditionals to each of the three frags. The challenge for this project was to balance sharp, stylish bands with smooth visual transitions. This challenge helped me gain a deeper understanding of model space and view space by accurately computing and propagating normal matrices.
